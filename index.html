About Me

Saya merupakan seorang laki-laki kelahiran Tangerang pada 1 Juli 1998, dan saya berusia 23 tahun. Saya beralamat di Jl. Pendidikan Batuceper Tangerang. Saat ini saya sedang menempuh pendidikan S1 jurusan Teknik Informatika di Universitas Bina Sarana Informatika kampus cabang Cengkareng.

Saya memiliki kesenangan dalam dunia Teknologi, khususnya di bidang bahasa pemrograman. Saya memiliki cita cita menjadi seorang Web Programmer terutama pada bagian FrontEnd Developer. Kedepannya, saya akan lebih serius lagi dalam menekuni bidang Teknologi ini dan selalu berusaha dan berdoa agar cita-cita saya dapat tercapai.
